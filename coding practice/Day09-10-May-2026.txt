Services from class (Day 9)
---------------------------

23. AWS Batch — batch jobs at scale
    - Job queues, compute environments; good for large offline processing
    - Can use Spot for cost; reads/writes often via S3

24. AWS Glue — serverless ETL
    - Crawlers discover schema; jobs transform data for analytics
    - Works with S3, Redshift, Spark-style jobs (Python/Scala)

25. EFS — shared file storage for Linux
    - NFS mount on multiple EC2; grows automatically
    - Content repos, shared config, some analytics workloads

26. Elastic Beanstalk — PaaS deploy
    - Upload app; AWS handles capacity, load balancer, health
    - Java, Python, Node, .NET, etc.; can wire RDS/S3 behind it

27. CloudFront — CDN
    - Edge caches static/dynamic content close to users
    - Origins: S3, ALB, custom; pairs with WAF/Shield

28. Snowball — physical data transfer
    - Ship appliance for huge datasets when network transfer is too slow
    - Snowball Edge adds local compute

29. CodePipeline — CI/CD orchestration
    - Stages: source → build → deploy; plugs into GitHub/CodeCommit

30. CodeBuild — managed builds
    - compile, test, produce artifacts; scales automatically

31. CodeDeploy — deployment automation
    - Rolling, blue/green, canary to EC2, Lambda, ECS
    - Zero-downtime goal when configured correctly

32. X-Ray — distributed tracing
    - Trace requests across Lambda, EC2, API Gateway
    - Find latency bottlenecks in microservices
