Services from class (Day 8)
---------------------------

12. AWS Shield — DDoS protection
    - Standard included; Advanced for bigger sites + DRT support
    - Protects CloudFront, Route 53, ELB

13. AWS WAF — web application firewall
    - Rules on ALB, API Gateway, CloudFront
    - Blocks SQLi, XSS, bad bots; can tie to Lambda for custom logic

14. GuardDuty — threat detection
    - ML + threat intel on VPC flow, CloudTrail, DNS logs
    - Findings for odd API calls, recon, exfiltration

15. Inspector — vulnerability scanning
    - Scans EC2, container images, Lambda for CVEs
    - Findings with severity; EventBridge for automation

16. KMS — encryption keys
    - CMKs for S3, EBS, RDS, etc.; rotation and IAM policies on keys
    - Usage logged in CloudTrail

17. Systems Manager (SSM) — operations hub
    - Patch Manager, Parameter Store, Run Command
    - Same family as Session Manager from Day 7

18. RDS — managed relational DB
    - MySQL, Postgres, MariaDB, Oracle, SQL Server
    - Automated backup, Multi-AZ failover, read replicas

19. Aurora — AWS-optimized SQL
    - MySQL/Postgres compatible; faster storage; Aurora Serverless option

20. DynamoDB — managed NoSQL
    - Key-value / document; on-demand or provisioned capacity
    - Single-digit ms latency at scale

21. Redshift — data warehouse
    - Columnar SQL analytics; Spectrum queries S3 data

22. DMS — database migration
    - Homogeneous/heterogeneous migrations with ongoing replication
    - On-prem Oracle/SQL Server → RDS/Aurora
